# ✨ Task 5: Dynamic Login Form Using JavaScript

This project showcases a **dynamic login form** that is fully created and styled using **JavaScript and CSS**, without any static HTML markup. The task demonstrates DOM manipulation, form validation, and responsive styling—all done programmatically.

✅ Completed as part of my internship at **Main Flow Services and Technologies Pvt. Ltd.**

---

## 📌 Features

- 🧱 Entire form (heading, inputs, button) generated via JavaScript
- 💡 Responsive and user-friendly layout using modern CSS
- ✍️ Real-time form validation with helpful user feedback
- 🔔 Alert-based display of login credentials on submission
- ❌ Prevents submission if fields are empty or invalid

---

## 🛠️ Tech Stack

| Area       | Technology      |
|------------|-----------------|
| Markup     | JavaScript DOM Manipulation |
| Styling    | CSS3            |
| Interaction | Vanilla JavaScript (ES6+) |
| Tools      | VS Code, Live Server |

---

## 🚀 Functionality

1. **JavaScript** dynamically creates:
   - A heading (`Login`)
   - Email & Password fields
   - A Submit (`Login`) button

2. CSS styles the form to:
   - Be centered on the screen (mobile & desktop)
   - Have padding, rounded corners, and clean visual hierarchy
   - Include interactive hover/focus effects

3. Event Listener:
   - Validates inputs on button click
   - Alerts the email & password values
   - Highlights fields if left empty
